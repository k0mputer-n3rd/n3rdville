Allen’s Yarn Crafts

This website began as a project for the Websites for Beginners class. Computers is one of my passions; another is yarn crafts. So it made sense to fulfill my class project requirements with something I enjoyed doing. Eventually this project may be expanded to the public to sell projects, patterns, etc.


Granny Squares

This is a basic granny square, using different colors for each row. There are many variations of granny squares, with many different sizes. Granny squares are usually stitched togther (with yarn) to make afghans (blankets made with yarn, usually knitted or crocheted), but can be used to make sweaters and other objects.
